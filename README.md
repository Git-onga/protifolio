# Bildad Gitonga | Personal Portfolio 👨🏽‍💻

A modern, responsive personal portfolio showcasing skills, projects, and contact information. Built with semantic HTML5, vanilla CSS, and JavaScript — featuring glassmorphism design, scroll-reveal animations, a typed-text hero effect, and a mobile hamburger menu.

## Deployment 🚀

The project is live at:

🔗 [https://git-onga.github.io/protifolio/](https://git-onga.github.io/protifolio/)

Hosted on GitHub Pages — auto-updates from the `main` branch.

## Features 🔢

- **Hero Section** — gradient orbs, typed-text role cycling, CTA buttons
- **About Section** — image with grayscale-to-color hover, bio text
- **Skills Section** — glassmorphism cards with scroll-reveal animations
- **Projects Section** — hover-zoom image cards with tech stack icons
- **Contact Section** — social links + styled contact form
- **Mobile Nav** — full-screen hamburger overlay with smooth transitions
- **Scroll Animations** — IntersectionObserver-powered reveal effects
- **Responsive Design** — mobile-first with breakpoints at 480px, 768px, 1024px

## Tech Stack ⚙️

- HTML5 (semantic)
- CSS3 (custom properties, glassmorphism, `@import` Google Fonts)
- Vanilla JavaScript (IntersectionObserver, typed text, hamburger menu)

## Usage ♻️

1. Clone or download the repository
2. Open `index.html` in your browser
3. All images and assets are in the `pics/` folder
